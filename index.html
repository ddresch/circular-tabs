circular-tabs

A lightweight jQuery plugin that renders an interactive circular tab interface with drag-and-drop reordering.

GitHub jQuery MIT License

Live Demo

Click the "+" button to add tabs. Drag any tab to reorder.

+

Features

Radial Layout

Tabs are arranged in a circle using CSS transforms with 15-degree spacing.

Drag & Drop

Grab any tab and drag it around the circle. Tabs swap positions automatically.

Snap to Position

On release, tabs snap smoothly to their nearest slot with a CSS transition.

Lightweight

Only ~110 lines of JavaScript. No build tools, no frameworks beyond jQuery.

Usage

Include jQuery, the plugin script, and the stylesheet, then add the markup:

<!-- Dependencies -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script src="circular-tabs.js"></script>
<link href="circular-tabs.css" rel="stylesheet">

<!-- Markup -->
<div id="circular-tabs">
  <div id="marker" class="circular-tab"
       data-moveable="false"
       onclick="addTab()">
    <span>+</span>
  </div>
  <div id="innerCircle"></div>
</div>

API

The plugin exposes global functions you can call directly:

// Add a new tab programmatically
addTab();

// Rotate a tab element to a specific CSS degree
rotateTab($(element), 45);

// Convert polar angle (atan2) to CSS degrees
var cssDegs = convertThetaToCssDegs(theta);
